About the Project

This app utilizes advanced AI models to support the detection and evaluation of dysarthria, a motor speech disorder. By processing speech recordings, we provide insight into severity and potential treatments.

  • Binary Classification: Detect if dysarthria is present
  • Severity Analysis: Classify the level of severity
  • English Feature Tools: Transcription, voice cloning, emotion detection
  • Russian Model: Clean speech generation using GANs
